# Contributing

Спасибо, что хочешь помочь проекту! Ниже - как устроен репозиторий и как в нём работать.

## Структура репозитория

```
.
├── lib/        # Python-библиотека golden-ration-ns (публикуется на PyPI)
└── frontend/   # веб-калькулятор на React + Pyodide (деплоится на GitHub Pages)
```

Это два независимых подпроекта в одном репозитории — у каждого своё окружение, зависимости и тесты.

## Работа с библиотекой (`lib/`)

```bash
cd lib
uv sync          # поставить зависимости, включая dev-группу (mypy, pytest)
```

Доступные команды (`make help` покажет тот же список):

```bash
make test        # прогнать тесты
make typecheck    # проверить аннотации (mypy)
make build        # собрать пакет
make clean        # удалить артефакты сборки
```

Тесты лежат в `lib/Tests/`, код — в `lib/src/goldenratio/`.

## Работа с фронтендом (`frontend/`)

```bash
cd frontend
npm install
npm run dev       # локальный сервер разработки
npm run build     # прод-сборка (проверяется tsc -b перед сборкой)
npm run lint       # eslint
```

## Стиль кода

- Отступы и переносы строк заданы в `.editorconfig` — большинство редакторов подхватывают его автоматически.
- Для Python — аннотации типов обязательны там, где это разумно, проверяются через `mypy` (`make typecheck` в `lib/`).
- Для фронтенда — `eslint` должен проходить без ошибок перед коммитом.

## Коммиты

Используем короткий префикс в стиле conventional commits: `feat:`, `fix:`, `docs:`, `style:`, `refactor:`, `test:`, `chore:`. Например:

```
fix: correct GitHub Pages deploy for frontend subfolder
```

Заметные изменения (особенно в `lib/`, влияющие на публикуемый пакет) стоит добавлять в [`CHANGELOG.md`](CHANGELOG.md), в раздел `Unreleased`.

## Ветки

- `main` — стабильная ветка
- `dev` — интеграционная ветка
- `feature/*` — рабочие ветки под конкретные задачи

PR открываем в `dev`, если не оговорено иное.

## Публикация библиотеки на PyPI

Делает мейнтейнер вручную после того, как версия в `lib/pyproject.toml` поднята:

```bash
cd lib
make publish   # uv publish, нужен PyPI-токен
```
